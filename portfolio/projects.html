Web Application with Auto Scaling and Load Balancer

This project involved designing and implementing a highly scalable web application architecture utilizing AWS services, such as:

  • Application Load Balancer: To distribute traffic.
  • Auto Scaling: To adjust instance capacity dynamically.
  • Launch Templates: For efficient provisioning.
  • CloudWatch: For monitoring performance and health.

The result was a cost-effective, secure, and highly available solution tailored to meet dynamic workload requirements.
